# Nowl (ノウル)
AIを活用した、資産運用のパートナーシステム。

## 🎯 主な目的
- 金融知識がなくても簡単に資産管理できる
- 家計や目標に連動した自動運用機能
- 市場や経済の動きをAIで分析・提案

## ⚙️ 技術スタック（予定）
- フロントエンド：React
- バックエンド：Spring Boot / FastAPI
- 分析エンジン：Python（Scikit-learn, TensorFlow）
- DB：PostgreSQL / TimescaleDB
- その他：Kafka, OAuth2.0, JWT, Keycloak
